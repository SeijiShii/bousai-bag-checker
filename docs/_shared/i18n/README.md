# i18n（横断設計）

多言語基盤。UI を 日本語(正本)/英語/中国語(簡体)/韓国語 の 4 言語で提供する。react-i18next + ロケールカタログ + ロケール検出(ブラウザ + 手動切替) + 言語切替 UI + 日付/数値整形。全 UI 文字列をキー化する。

## 対応言語
- `ja` (ソース/正本)、`en`、`zh-Hans`、`ko`

## 翻訳スコープ
- **対象**: UI チローム = ラベル / 見出し / ボタン / placeholder / aria-label / 空状態 / エラー / 入口説明(O41) / StatusChip / ナビ / 設定 / フィードバック
- **対象外**: 法務長文(プライバシー/利用規約/特商法) = **JA 正本**(各ページ「正本は日本語」注記、zh/ko/en は JA フォールバック)。コメント/型/ログ/テスト/開発者向け文字列。
